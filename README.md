# 🚀 Smart Career Recommendation System

## 📌 Overview
A full-stack web application that recommends career paths based on user skills, interests, and academic performance using Machine Learning.

---

## ✨ Features
- 🔐 User Authentication (Login/Signup)
- 🤖 Career Prediction using ML (Decision Tree)
- 📊 Dashboard with prediction history
- 📈 Data visualization using Chart.js

---

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Machine Learning: Scikit-learn
- Database: MySQL

---

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
